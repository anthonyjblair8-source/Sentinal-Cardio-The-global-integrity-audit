# Security Policy

## Supported Versions

The following versions of Sentinal-Cardio are actively supported with security updates:

| Version | Supported |
|---------|-----------|
| 5.1.x | :white_check_mark: |
| 5.0.x | :x: |
| 4.0.x | :white_check_mark: |
| < 4.0  | :x: |

## Security Philosophy

Sentinal-Cardio is built with a strong foundation in security. We adhere to the principles of least privilege, deterministic access, and immutable audit trails. Any deviation from these principles is treated as a security issue.

## Reporting a Vulnerability

If you find a vulnerability, please report it via GitHub Security Advisories or email us at `security@sentinal-cardio.example`. We follow a coordinated disclosure process and will acknowledge reports within 48 hours.
